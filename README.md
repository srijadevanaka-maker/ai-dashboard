# 🚀 AI Analytics Dashboard

A modern AI-powered analytics dashboard built using React.  
This project simulates real-time insights such as user activity, API performance, and model accuracy with interactive charts.

---

## ✨ Features

- 🔐 Login Page (basic navigation)
- 📊 Dashboard with key metrics
- 📈 Interactive charts (user activity, API usage)
- 🤖 Model performance visualization
- 📱 Responsive design

---

## 🛠 Tech Stack

- React.js
- React Router
- Tailwind CSS
- Recharts
- Axios

---

## 📊 Dashboard Overview

The dashboard includes:
- User activity tracking
- API usage metrics
- Model accuracy visualization
- Mock system data

---

## ▶️ How to Run

1. Clone the repository:
2. Go to project folder: cd ai-dashboard
3. Install dependencies: npm install
4. Start the project: npm start
